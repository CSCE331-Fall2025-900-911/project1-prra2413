Personal Projects

News Article Summarization with Transformers

Fine-tuned BART and Longformer encoder decoder transformers for abstractive news summarization, leveraging self-attention for summary generation and cross-attention for contextualization. Analyzed token length distributions and applied input/output truncation to balance context retention with efficient transformer training. Processed a multi-GB dataset, and optimized GPU training through gradient checkpointing, gradient accumulation, and FP16.


Sleep Disorder Classifier (XGBoost)

Developed an XGBoost algorithm to predict sleep disorders using metrics such as sleep duration, quality, BMI, blood pressure, and heart rate, achieving 92% accuracy. Applied data preparation techniques, including normalization, label and one-hot encoding, and feature selection, to enhance model performance.

Github: https://github.com/pranavprashantha/MachineLearning_SleepAnalysis


9x9 Sudoku Solver - Z3 SMT Solver

Applied predicate logic to define Sudoku constraints using compound predicates. Developed a Python script to generate SMT-Lib code, enabling the Z3 SMT solver to efficiently solve any 9x9 Sudoku.

Github: https://github.com/pranavprashantha/SMT_Sudoku_Solver
